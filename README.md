# Safety Connect

## Overview
Safety Connect is a smart workplace safety solution designed to improve worker protection in industrial environments. The project combines a mobile application with smart safety concepts to help reduce workplace accidents and improve emergency response.

## Features
- Worker safety monitoring
- Emergency alerts
- Incident reporting
- Smart helmet integration concept
- User-friendly interface

- UI/UX Design
